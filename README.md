# 7COM1079 Team Research and Development Project

## Research Question
Is there a relationship between flight duration and flight price?

## Description
This project analyses flight data using R to examine the relationship between flight duration (minutes) and flight price (INR).

## Method
- Data cleaned in R  
- Duration converted to minutes  
- Pearson correlation test used  
- Scatter plot and histogram created  

## Result
No statistically significant correlation was found between flight duration and flight price.
